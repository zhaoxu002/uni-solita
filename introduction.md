# 团购群主的专属小程序

功能如下
### 可配置主题色等外观表现

### 配置商品
  - 图片
  - 介绍
  - 价格
  - 库存
  - 特别推荐/置顶

### 提货点
  - 可配置多个供用户选择

### 配置团购并转发至群聊
  - 类目
  - 包含多个商品
  - 介绍
  - 将多个商品加入购物篮
  - 可配置开始/结束时间

### 下单
  - 用户可选择提货点
  - 昵称
  - 电话
  - 备注
  - 商品数量及价格
  - 联系客服

### 已订购
  - 用户可查询已订购的商品信息

### 订单信息导出
  - owner可按提货点分类导出订单信息
